# 🥙 Döner Station - Premium Gourmet Doner & Kebab Landing Page

Welcome to **Döner Station**! This is an ultra-modern, high-converting, and premium responsive web template custom-built for gourmet traditional doner restaurants, boutique kebab houses, and fast-casual food brands. It bridges traditional tastes with high-end modern digital design.

🔗 **[Live Demo Hosted on GitHub Pages](https://merveayliz.github.io/doner/)**




## ✨ Outstanding Core Features

* **📱 Flawless Mobile-First Fluid UX:** Deeply optimized for smartphone screens with a sleek grid layout. Zero overlapping elements, ensuring hungry customers can browse the menu seamlessly on the go.
* **⚡ Blazing Fast Architecture:** Built with highly lightweight, modern frontend engineering methodologies ensuring lightning-fast rendering speeds and flawless component performance.
* **🎯 High-Converting Call To Actions (CTA):** Strategically placed "Order Now" and interactive contact mechanisms designed to turn simple website traffic into high-value restaurant orders instantly.
* **🥘 Elegant Menu Presentation:** A highly visual, beautiful catalog system that highlights gourmet meats, traditional wraps, and traditional side dishes with crisp typography and clean card physics.
* **🎨 Aesthetic Dark & Warm Theme:** Specially curated color palettes matching hot grill tones with luxury modern web design to make the food photography pop beautifully.
